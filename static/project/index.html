Noteify

Noteify is a web app designed to save and serve text data in the form of notes. Noteify was developed in a simulated scrum environment modeling the Agile development lifecycle. It's functionality demonstrates client-server computing, client authorization, account creation and authentication with the help of Firebase, and much more.

This project was developed by a team of 3 over the course of a few weeks as an end-of-semester project in a Software Engineering course (Spring 2022).

    Features:

  • Create an account with a username and password
  • Verify account authenticity via email verification link
  • Automatically proceed into site once link is clicked
  • Only access pages if logged in and/or verified
  • Enter and save notes to view or delete later via real-time database

    • Note: Firebase trial has ended, disabling main functionality of website.


Accupay

Accupay is a lightweight GUI tool that calculates the after-tax paycheck of hourly workers. It asks users for information including: pay rate, hours worked, state of residence, and tax bracket. It demonstrates primary Object-Oriented Programming principles and industry best practices.

This project was developed over a few weeks as an end-of-semester project in an Object-Oriented Programming course (Spring 2021).

    Features:

  • Lightweight, easy-to-use GUI interface
  • Calculates Paycheck with given pay, hours, and tax information
  • Allows user to reset fields

    • Note: Tax bracket data no longer accurate.
